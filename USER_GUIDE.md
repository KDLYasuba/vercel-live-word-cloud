# ワードクラウド 操作ガイド

このガイドは、イベントや研修でワードクラウドを利用する方向けの操作説明です。

## できること

- 参加者がスマホやPCから短いワードを投稿できます
- 投稿されたワードを会場スクリーンにリアルタイム風に表示できます
- 司会者は表示タイトル、表示モード、参加受付を操作できます
- 管理者URLには利用期限があります

## 画面の種類

| 画面 | 使う人 | 用途 |
| --- | --- | --- |
| 司会者画面 | 司会者・運営担当者 | タイトル変更、表示モード切替、受付停止など |
| 参加者入力画面 | 参加者 | ワード投稿 |
| 表示専用画面 | 会場スクリーン・プロジェクター | 投稿結果の表示 |

## 司会者向け

### 1. 司会者URLを開く

運営担当者から共有された司会者URLを開きます。

```text
https://.../admin?token=...
```

このURLはイベントごとの管理用URLです。第三者に転送しないでください。

### 2. タイトルを設定する

「タイトル」欄に、会場に表示したいテーマを入力します。

例:

```text
今日の研修で印象に残ったこと
```

入力後、「タイトルを適用」を押します。

注意:

- タイトルを適用すると、画面上の表示対象はリセットされます
- 過去の投稿データは物理削除されません
- 参加者URLや表示専用URLは変わりません

### 3. 表示モードを選ぶ

司会者画面の「表示モード」で切り替えできます。

| ボタン | 内容 |
| --- | --- |
| Goワード | 投稿文を単語に分解し、よく出た単語を大きく表示 |
| 原文に戻す | 参加者が投稿した文言をそのまま表示 |

ワークショップで傾向を見たい場合は「Goワード」、投稿内容そのものを見たい場合は「原文に戻す」を使います。

### 4. 表示専用画面を開く

「表示専用を開く」を押すと、会場スクリーン用の画面が別タブで開きます。

この画面をプロジェクターや大型モニターに表示してください。

### 5. 参加者入力画面を開く

「参加者入力を開く」を押すと、参加者が投稿する画面を確認できます。

表示専用画面の右上には参加者用QRコードも表示されます。参加者にはこのQRコードを読み取ってもらうとスムーズです。

### 6. 参加受付を止める・再開する

「受付を停止」を押すと、参加者からの新規投稿を止められます。

受付停止中:

- 参加者は投稿できません
- 表示済みのワードは残ります

再開したい場合は「受付を再開」を押します。

### 7. 利用期限について

管理者URLには終了期限があります。

期限を過ぎると:

- 司会者画面からの操作ができなくなります
- 参加者からの投稿ができなくなります
- 表示専用画面は閲覧できる場合がありますが、新規投稿は反映されません

期限を延長したい場合は、運営担当者に新しいURLの発行を依頼してください。

### 8. CSVデータを出力する

司会者画面の一番下にある「CSVデータを出力」を押すと、現在の表示対象になっている投稿をCSVで保存できます。

CSVには次の3列が出力されます。

| 列 | 内容 |
| --- | --- |
| タイトル | 司会者画面で設定されている現在のタイトル |
| ワード | 参加者から投稿された内容 |
| 時間 | 投稿日時 |

CSVはShift-JIS形式です。Windows版Excelでも文字化けしにくい形式です。

注意:

- タイトル変更前など、リセット前の過去投稿はDBには残りますが、このCSVには出力されません
- CSV出力は、管理者URLの終了期限から3日後まで利用できます
- 3日を過ぎると、司会者URLからのCSV出力はできなくなります

## 参加者向け

### 1. 参加者入力画面を開く

司会者または運営担当者から共有されたURLを開きます。

表示専用画面にQRコードが表示されている場合は、スマホで読み取ってください。

### 2. 現在のタイトルを確認する

入力画面には、投稿先のタイトルが表示されます。

例:

```text
現在のタイトル：
今日の研修で印象に残ったこと
```

### 3. ワードを入力する

「ワード」欄に投稿したい内容を入力します。

入力できる文字数は30文字以内です。

例:

```text
チームで考える大切さ
```

### 4. 送信する

「送信」を押すと投稿されます。

送信後、表示専用画面に反映されます。反映まで数秒かかることがあります。

### 5. 投稿できない場合

以下の場合は投稿できません。

- イベントの受付が停止されている
- イベントの利用期限が過ぎている
- 入力が空欄
- 31文字以上入力している
- 短時間に大量投稿している

その場合は、司会者または運営担当者に確認してください。

## 表示専用画面について

表示専用画面は、会場スクリーンや大型ディスプレイに表示するための画面です。

特徴:

- 数秒ごとに投稿内容を更新します
- 2分間更新がない場合は待機状態になります
- 新しい投稿があると更新を再開します
- 右上のQRコードから参加者入力画面を開けます
- QRコードをクリックすると拡大表示できます

## 投稿時の注意

参加者には、以下を事前に案内してください。

- 個人情報を書かない
- 他人を傷つける言葉を書かない
- 機密情報を書かない
- イベントテーマに沿った内容を投稿する
- 連続投稿しすぎない

## よくある質問

### タイトルを変更するとURLも変わりますか？

変わりません。

タイトルは画面に表示される名前です。参加者URLや表示専用URLはそのまま使えます。

### タイトルを変更すると投稿は消えますか？

画面上はリセットされますが、データベースから物理削除はされません。

タイトル適用後は、その時点以降の投稿だけが表示対象になります。

### 同じタイトルをもう一度使えますか？

使えます。

同じタイトルで「タイトルを適用」すると、画面上の投稿はリセットされ、その後の投稿だけが表示されます。

### 参加者は何文字まで投稿できますか？

30文字までです。

### 期限が切れたらどうなりますか？

司会者操作と参加者投稿が停止します。継続利用する場合は、新しい管理者URLを発行してもらってください。

## トラブル時の確認

### 投稿が表示されない

- 参加者入力画面と表示専用画面が同じイベントURLか確認してください
- 司会者画面で受付停止になっていないか確認してください
- 表示専用画面を再読み込みしてください
- 利用期限が切れていないか確認してください

### QRコードが読み取れない

- 表示専用画面のQRコードをクリックして拡大してください
- 画面の明るさを上げてください
- 参加者URLをチャットなどで直接共有してください

### 司会者画面が操作できない

- 管理者URLが正しいか確認してください
- 利用期限が切れていないか確認してください
- URLを途中で改変していないか確認してください
